Kenji Mode
Are you ready for Kenjius? 1. Press the Play button. Genius will give the first signal. Repeat the signal by pressing the same color lens.
2. Genius will duplicate the first signal and add one. Repeat these two signals by pressing the same color lenses, in order.
3. Genius will duplicate these first two signals and add one.
4. Continue playing as long as you can repeat each sequence of sig- nals correctly. When set to medium (which is default speed) or fast speed, Genius speeds up after each round.
5. If you fail to repeat a sequence exactly, or if you take more than 5 seconds (in normal speed, 10 seconds in slow speed or 2 seconds in fast speed) to repeat a signal, Genius responds with a "MEU DEUS" sound. This means you've lost, and the sequence of signals ends.
6. For an extra challenge, try KENJI MODE. In this mode (which is default mode), you must start over if you make any mistake, or run out of time.
